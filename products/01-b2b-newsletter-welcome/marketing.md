## Long description (for the listing page)

Most B2B newsletters leak 40-60% of new readers in the first 7 days. The cause is almost always a generic auto-responder that confirms the subscription and does nothing else.

The fix is a 5-email welcome sequence that delivers value, establishes credibility, gets a small action, tells a story, and - on day 10 - makes a soft conversion ask.

The problem is that writing this sequence takes a week. The prompts in this bundle let you write it in an hour.

**What you get:**

- 5 prompt templates, one per email in the sequence (delivery, proof, quick win, origin story, soft CTA)
- 12 personalization variables you fill once and reuse
- 2 fully worked examples - a B2B devtools newsletter and a B2B fintech newsletter - so you can see what good output looks like
- A 10-rule anti-AI-tone checklist so the output doesn't read like ChatGPT wrote it
- A commercial license: use the output forever, in your own business, no attribution required

**What this is not:**

- Not a generic "10 ChatGPT prompts" pack
- Not a tool - the prompts work with any LLM you already use
- Not for B2C newsletters (the voice is wrong for that)
- Not a magic button - you still need to fill in the variables and run the output through the checklist

**Who this is for:**

- Solo founders and small marketing teams running a B2B newsletter
- 200-10,000 subscribers
- Newsletter is tied to a SaaS product, paid community, or service

**Time to first sequence:** ~60 minutes if your variables are sharp, ~3 hours if you need to think about them.

**Lifetime updates included.**
